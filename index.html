Problem & Solution

GA 현장의 실제 문제를
정면으로 해결합니다

  • 😤
    보험사마다 다른 상품명·구조
    같은 종신보험도 회사별로 특약 구조가 달라 모집인이 직접 비교표를 만드는 데 수 시간이 소요됩니다.
  • 📊
    실시간 보험료 시뮬레이션 불가
    고객 나이·건강 상태에 따른 실시간 보험료 계산 없이 영업이 이뤄져 고객 신뢰도가 낮아집니다.
  • 📄
    고객 맞춤 설명 자료 제작 과다
    한 명의 고객을 위해 여러 보험사 자료를 취합·편집하는 반복 작업이 모집인의 생산성을 크게 저하시킵니다.
  • 🔗
    GA 채널 외부 이탈 통제 어려움
    모집인이 외부 채널로 고객을 유도하거나 GA 모르게 계약을 처리하는 경로를 구조적으로 차단하기 어렵습니다.

How It Works

역할에 따라 다르게 작동합니다

GA 관리자·모집인·고객 3가지 역할에 맞게 최적화된 워크플로우를 제공합니다.

01
비교 대시보드에서 상품 선택
종류·보험사·보장 항목 필터로 비교할 상품(최대 4개)을 드래그&드롭으로 선택합니다.
02
보험료 계산기로 고객 조건 입력
고객의 성별·나이·납입기간을 슬라이더로 조정해 각 상품의 표준 가이드 보험료를 즉시 확인합니다.
03
AI 추천 분석 실행
고객 프로필을 입력하면 LLM이 가중치 점수를 계산하고 상품별 맞춤 추천 이유를 자연어로 작성합니다.
04
제안서 생성 & 카카오 공유
고객 이름이 개인화된 브랜드 제안서를 생성해 카카오톡 링크 또는 QR코드로 즉시 공유합니다.
05
고객 이동 및 상담 요청 알림 수신
고객이 제안서 링크를 열면 실시간 알림을 받고, "가입 상담 요청" 시 카카오알림톡이 즉시 발송됩니다.

01
카카오·문자로 링크 수신
모집인이 공유한 개인화 링크를 스마트폰에서 클릭합니다.
02
AI 맞춤 분석 배너 확인
제안서 상단의 "AI가 내 조건으로 분석한 추천 상품" 배너로 어떤 기준으로 추천됐는지 먼저 이해합니다.
03
상품 카드 비교 & AI 추천 이유 확인
항목별 점수바와 LLM이 작성한 맞춤 추천 이유 2~3문장을 읽으며 상품을 직관적으로 비교합니다.
04
원하는 상품 "가입 상담 요청"
관심 상품 카드 하단의 버튼을 눌러 이름·연락처·희망 연락 시간을 Bottom Sheet에서 간편 입력합니다.
05
담당 모집인 정보 확인 & 연결
요청 완료 화면에서 담당 모집인 이름·프로필·예상 연락 시간을 확인하고 카카오채널로 바로 연결할 수 있습니다.

모델 교체 자유 — LLM_PROVIDER 환경변수 하나로 OpenAI / Anthropic / Google 간 즉시 전환 가능합니다.
